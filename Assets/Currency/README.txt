Australian Currency Assets
==========================

Place your Australian banknote and coin images in this folder.
The application will use these images on the denomination buttons in the Buy-In view.

Expected File Names:
-------------------

NOTES (PNG format, recommended size: 200x100 pixels):
- note_100.png  - $100 note (Green - Dame Nellie Melba)
- note_50.png   - $50 note (Yellow - David Unaipon / Edith Cowan)
- note_20.png   - $20 note (Red - Mary Reibey / John Flynn)
- note_10.png   - $10 note (Blue - Dame Mary Gilmore / Banjo Paterson)
- note_5.png    - $5 note (Pink - Queen Elizabeth II / Parliament House)

COINS (PNG format, recommended size: 100x100 pixels):
- coin_2.png    - $2 coin (Gold - Aboriginal elder)
- coin_1.png    - $1 coin (Gold - Five kangaroos)
- coin_50c.png  - 50 cent coin (Silver - Coat of Arms)
- coin_20c.png  - 20 cent coin (Silver - Platypus)
- coin_10c.png  - 10 cent coin (Silver - Lyrebird)
- coin_5c.png   - 5 cent coin (Silver - Echidna)

Notes:
------
- Use PNG format with transparency for best results
- Images will be scaled to fit the buttons
- If an image is missing, a colored placeholder will be shown
- For best appearance, use images with similar aspect ratios to the recommended sizes
